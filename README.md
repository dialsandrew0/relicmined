# RelicMined - God Tier Object Intelligence Platform

## Overview
RelicMined is an AI-powered platform for object identification, research, valuation, and sales strategy from single photos. Built with computer vision, multi-agent systems, and opportunity engine.

## Quick Start
pip install -r requirements.txt
python main.py

## Structure
- cv_pipeline/: Computer vision orchestrator
- agents/: Multi-agent authentication
- opportunity_engine/: Sales strategies
- docs/: Full vision and specs
- ui/: Frontend mock with Streamlit

God Tier features: Single-photo input, Evidence Graph, Trash-to-Elite playbooks.
